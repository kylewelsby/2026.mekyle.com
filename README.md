Most portfolios show you the finished meal. This one lets you watch the cooking.

## ğŸ§  What This Repo Demonstrates

**Spec-Driven Development** using [OpenSpec](./openspec/AGENTS.md): writing detailed specifications before touching code. This prevents the classic mistake of building the wrong thing really efficiently.

**AI-Assisted Development** with transparent prompting. The `CLAUDE.md` file contains every instruction given to AI tools. No magic, no hidden context: just clear direction.

**Iterative Design Thinking** through change proposals that capture *why* decisions were made, not just *what* was decided.
